Design Notes

  • Cost is how much chakra a single clone drains relative to others.
  • Durability is the rough hit tolerance before the clone breaks.
  • Contact Effect is a status or burst applied when a target touches the clone.
  • Environment toggles modify stats (e.g., Wet boosts Water; Stormy boosts Lightning).
